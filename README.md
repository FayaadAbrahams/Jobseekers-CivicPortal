# Civic Portal - Jobseekers

> **1st Place** | City of Cape Town Hackathon 2026

A web-based portal built in a 48-hour hackathon sprint to address a pressing community challenge through an innovative digital solution. Built with **Vite**, **TypeScript**, **HTML**, and **CSS**.

---

## Project Overview

This project was built during a City of Cape Town-hosted hackathon, where teams were challenged to address a pressing issue within local communities through innovative digital solutions. As team lead, I guided a group of interns through the full lifecycle — brainstorming, prototyping, and developing a functional web application within a tight 48-hour timeframe.

Our team's solution stood out for its practical impact and strong execution, earning **first place among all competing teams**.

## The Scope

To build a solution that assists with poorer individuals who had no access to cellphones or computers in their personal usage.

## The Solution

We conceptualized and built a functional web-based portal designed to balance usability, accessibility, and real-world impact for the target community. Key aspects of the build:

- **Intuitive navigation** for fast, low-friction access to core functionality
- **Real-time user feedback components** to keep users informed as they interact with the portal
- **Responsive design** ensuring usability across devices
- Rapid planning and review cycles to adapt the solution as new insights emerged during the sprint

## 🛠️ Tech Stack

- **[Vite](https://vitejs.dev/)** — build tooling and dev server
- **TypeScript** — application logic and dynamic interface behavior
- **HTML5** — front-end structure
- **CSS3** — styling and responsive layout


## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later recommended)
- npm (comes with Node.js) or yarn

### Installation

```bash
# Clone the repository
git clone <repo-url>
cd <project-folder>

# Install dependencies
npm install
```

### Running locally

```bash
npm run dev
```

The app will be available at `http://localhost:3000` by default.

### Building for production

```bash
npm run build
npm run preview
```
